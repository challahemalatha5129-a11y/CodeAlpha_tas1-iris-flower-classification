# Iris Flower Classification

This project uses Machine Learning to classify iris flowers into:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Seaborn

## Features
- Dataset loading
- Model training
- Accuracy evaluation
- Flower prediction
- Data visualization

## Algorithm Used
K-Nearest Neighbors (KNN)

## Output
The model predicts flower species based on flower measurements.
